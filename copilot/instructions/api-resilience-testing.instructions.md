# API Resilience Testing Skill

Tests REST/HTTP APIs beyond the happy path — negative, fuzz, contract, and
security testing — to find critical failures before production. Use when adding
or changing an endpoint, reviewing an API PR, writing API tests, or designing
request/response schemas.
