## [1.0.0] - 2026-07-01

### Added

- Canonical `skills/` directory following the open Agent Skills standard — every skill is a
  self-contained folder (`skills/<name>/SKILL.md` + optional `references/`), installable by
  `npx skills add solvelab/ai-skills` across 70+ agents.
- Claude Code plugin + marketplace manifests (`.claude-plugin/plugin.json`,
  `.claude-plugin/marketplace.json`): install with `/plugin marketplace add solvelab/ai-skills`.
- Automated releases: semantic-release on every push to `master`
  (`.github/workflows/ci.yml`) — version computed from Conventional Commits, propagated
  to `VERSION` and the plugin manifests by `scripts/set-version.sh`, changelog and GitHub
  Release generated, `vX.Y.Z` tagged.
- CI validation (`.github/workflows/ci.yml`): wrapper sync, version coherence,
  frontmatter checks.

### Changed

- `shared/skills/*/content.md` merged into `skills/<name>/SKILL.md`; `game/r3f-*` flattened
  to `skills/r3f-*`.
- `claude/`, `codex/`, `cursor/`, `copilot/` wrappers are now generated from `skills/` by
  `generate.sh` (kept for backward compatibility with existing installs).
- `install.sh` defaults to native symlink install into `~/.claude/skills/` (no `CLAUDE.md`
  edit needed); `--legacy` keeps the old CLAUDE.md block behavior.
